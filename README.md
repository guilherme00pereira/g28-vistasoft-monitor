# Vistasoft Monitor

### Orientações

Remoção de imóveis duplicados e obsoletos
Passo 01
- listar todos os códigos cadastrados no site

Passo 2
- Listar todos os códigos marcados mostrar no site no vista crm

Passo 3
Executar as seguintes ações
- Imóvel que esteja no site, mas não esteja na lista do vista para "mostrar no site" devem ser apagados do site (apagar também as imagens dos imóveis)
- Imóvel que esteja no vista crm marcado "mostrar no site", mas não esteja no site Deve ser cadastrado no site
- Ter opção de desativar função
- Mostrar tela de log das ações
- Varredura e ações devem acontecendo diariamente

Quando o imóvel vier sem o campo "exibir no site" marcado, deve ser excluído do site (com imagens)
